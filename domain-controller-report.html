一、调研背景与目标

随着汽车电子电气（E/E）架构从分布式向集中式演进，域控制器成为智能汽车的核心竞争高地。行业正在经历三个关键转变：

①
架构集中化
分布式 → 域集中 → 中央计算平台(CCU)
②
舱驾融合化
分离式 → One Board → One Chip
③
芯片自主化
英伟达/高通 → 自研芯片
⚠️ 不确定性声明：部分信息来源于公开渠道，具体技术参数和规划时间线可能因车型配置、市场策略调整而变化。自研芯片的实际量产时间存在较大不确定性。

3.1 比亚迪 — 璇玑架构 · 天神之眼ABC · 璇玑A3芯片规划中
维度详情
EE架构璇玑架构（2024.1发布），"一脑、两端、三网、四链"
座舱方案DiLink智能座舱系统；高端（腾势/仰望）高通8295；中低端高通8155/6125
座舱OSDiLink OS（基于Android深度定制）
智驾方案天神之眼A（三激光+双Orin-X，仰望） / B（激光+单/双Orin-X，腾势/汉唐） / C（纯视觉+征程5/J6M，王朝/海洋）
智驾芯片英伟达Orin-X（高阶）、地平线征程5/J6M（中低阶）
域控架构分离式 已申请舱驾融合专利 CN121763694A

自研芯片璇玑A3（4nm智驾芯片，已流片）；座舱芯片在研
📌 判断：比亚迪"量大面广"策略清晰——高端用英伟达+高通，中低端用地平线+高通。舱驾一体是远期方向（2027+），短期内分离式架构仍为主力。

七、趋势判断与机会洞察

五大确定性趋势

趋势判断置信度
舱驾一体是确定方向2026-2028年将成为量产集中期⭐⭐⭐⭐⭐
智驾平权加速下沉10-15万级车型将普及高速NOA⭐⭐⭐⭐⭐

双极格局松动地平线在中阶、自研芯片在高阶形成替代⭐⭐⭐⭐
软件定义硬件整车OS成为差异化关键（蔚来天枢/小米HyperOS/鸿蒙）⭐⭐⭐⭐

风险提示

风险说明
自研芯片量产风险从流片到上车通常需2年+，良率和产能是重大不确定性
舱驾一体技术复杂度功能安全（智驾ASIL-D + 座舱QM）的融合是巨大挑战
供应链锁定效应深度绑定单一芯片供应商（如华为/高通）可能限制灵活性
算力军备竞赛芯片算力快速增长，过早量产可能面临快速过时风险

机会判断

  1. 中阶智驾芯片国产替代：地平线征程6系列在10-20万车型市场有巨大空间
  2. 座舱AI芯片：大模型上车推动座舱算力需求激增（理想马赫M100路径值得关注）
  3. 舱驾融合中间件：跨域通信、安全隔离、算力调度中间件是软件层机会
  4. 整车OS生态：蔚来天枢/小米HyperOS/鸿蒙三种路径各有优势，生态绑定是关键壁垒
