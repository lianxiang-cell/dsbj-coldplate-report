# 国内主流车企智能座舱与智能驾驶域控制器发展现状及规划报告

> **报告类型**：竞品分析 + 技术趋势调研  
> **调研时间**：2026年6月  
> **数据截止**：2025年量产车型 + 2026年已公布规划  
> **目标受众**：产品规划、技术战略、管理层决策  

---

## 一、调研背景与目标

### 1.1 调研背景

随着汽车电子电气（E/E）架构从分布式向**集中式**演进，域控制器成为智能汽车的核心竞争高地。行业正在经历三个关键转变：

1. **架构集中化**：从"一功能一ECU"的分布式架构 → 域集中式架构 → 中央计算平台（CCU）
2. **舱驾融合化**：座舱域与智驾域从完全分离 → One Board（同一板卡）→ One Chip（单芯片）
3. **芯片自主化**：从依赖英伟达/高通 → 自研芯片（蔚来神玑、小鹏图灵、比亚迪璇玑）

### 1.2 调研目标

| 维度 | 核心问题 |
|------|----------|
| 座舱方案 | 各车企主流/旗舰车型采用什么座舱SoC和操作系统？ |
| 智驾方案 | 采用什么智驾芯片、传感器配置和算法方案？ |
| 域控架构 | 当前是分离式还是舱驾一体？未来规划如何？ |
| 供应链格局 | 自研 vs Tier1供应商（英伟达、高通、地平线、黑芝麻等） |

### 1.3 调研方法

- 公开资料收集（车企官网、行业报告、拆解分析）
- 专利/公告信息（国家知识产权局）
- 行业媒体报道与分析师报告
- CES/车展等展会公开信息

> ⚠️ **不确定性声明**：部分信息来源于公开渠道，具体技术参数和规划时间线可能因车型配置、市场策略调整而变化。自研芯片的实际量产时间存在较大不确定性。

## 二、核心发现（Executive Summary）

### 2.1 行业全景判断

1. **分离式仍是主流**：当前90%以上量产车型采用座舱+智驾分离式域控架构
2. **舱驾一体是确定性趋势**：2025-2026年成为关键窗口期，多家车企已公布量产时间表
3. **芯片自主化加速**：蔚来（神玑5nm）、小鹏（图灵）、比亚迪（璇玑A3 4nm）均已流片/量产
4. **"智驾平权"驱动下沉**：比亚迪天神之眼C方案将智驾下探至10万级，地平线征程6系列成为中低阶主力

### 2.2 关键发现

| 发现 | 详细说明 |
|------|----------|
| 高通8295成座舱标配 | 8295/8295P是2024-2025年旗舰车型座舱首选芯片 |
| 英伟达Orin-X主导高阶智驾 | 双Orin-X（508 TOPS）是高阶智驾主流方案 |
| 地平线征程系列快速崛起 | 征程5/6M/6P在中阶智驾市场渗透率极高 |
| 零跑已实现四域合一 | 四叶草架构将座舱/智驾/动力/车身融合为中央计算平台 |
| 蔚来率先量产舱驾融合 | 2024款二代平台已升级为ADAM中央计算平台（One Board） |

## 三、各车企智能座舱与智驾域控详细分析

### 3.1 比亚迪

| 维度 | 详情 |
|------|------|
| **EE架构** | 璇玑架构（2024.1发布），"一脑、两端、三网、四链" |
| **座舱方案** | DiLink智能座舱系统；高端车型（腾势/仰望）采用高通8295；中低端采用高通8155/6125 |
| **座舱OS** | DiLink OS（基于Android深度定制） |
| **智驾方案** | **天神之眼**分级体系： |
|  | - **天神之眼A（三激光版）**：3激光雷达 + 双Orin-X，搭载仰望品牌 |
|  | - **天神之眼B（激光版）**：1-2激光雷达 + 单/双Orin-X，搭载腾势/汉唐高配 |
|  | - **天神之眼C（视觉版）**：纯视觉 + 地平线征程5/J6M，搭载王朝/海洋系列 |
| **智驾芯片** | 英伟达Orin-X（高阶）、地平线征程5/J6M（中低阶） |
| **域控架构** | **分离式**（座舱域+智驾域独立）；已申请舱驾融合域控制器专利（CN121763694A，2026.3公开） |
| **舱驾一体规划** | 自研璇玑A3芯片（4nm，单颗>700 TOPS）预计2027年首搭腾势新车；舱驾融合专利已公开 |
| **自研芯片** | 璇玑A3（4nm智驾芯片，已流片）；座舱芯片在研 |
| **供应商** | 座舱：高通 + 自研；智驾：英伟达/地平线 + 自研芯片规划中 |

> 📌 **关键判断**：比亚迪当前"量大面广"策略清晰——高端用英伟达+高通，中低端用地平线+高通。舱驾一体是远期方向（2027+），短期内分离式架构仍为主力。

## 七、趋势判断与机会洞察

### 7.1 五大确定性趋势

| 趋势 | 判断 | 置信度 |
|------|------|--------|
| **舱驾一体是确定方向** | 2026-2028年将成为量产集中期 | ⭐⭐⭐⭐⭐ |
| **智驾平权加速下沉** | 10-15万级车型将普及高速NOA | ⭐⭐⭐⭐⭐ |
| **芯片自主化不可逆** | 头部车企自研芯片从智驾切入，逐步覆盖座舱 | ⭐⭐⭐⭐ |
| **高通+英伟达双极格局松动** | 地平线在中阶、自研芯片在高阶形成替代 | ⭐⭐⭐⭐ |
| **软件定义硬件** | 整车OS成为差异化关键（蔚来天枢/小米HyperOS/鸿蒙） | ⭐⭐⭐⭐ |

### 7.2 风险提示

| 风险 | 说明 |
|------|------|
| **自研芯片量产风险** | 从流片到上车通常需2年+，良率和产能是重大不确定性 |
| **舱驾一体技术复杂度** | 功能安全（智驾ASIL-D + 座舱QM）的融合是巨大挑战 |
| **供应链锁定效应** | 深度绑定单一芯片供应商（如华为/高通）可能限制灵活性 |
| **算力军备竞赛** | 芯片算力快速增长，过早量产可能面临快速过时风险 |

### 7.3 机会判断

1. **中阶智驾芯片国产替代**：地平线征程6系列在10-20万车型市场有巨大空间
2. **座舱AI芯片**：大模型上车推动座舱算力需求激增（理想马赫M100路径值得关注）
3. **舱驾融合中间件**：跨域通信、安全隔离、算力调度中间件是软件层机会
4. **整车OS生态**：蔚来天枢/小米HyperOS/鸿蒙三种路径各有优势，生态绑定是关键壁垒
